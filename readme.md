# Pet Management API

## Descripción
Esta API permite gestionar información sobre mascotas. Proporciona funcionalidades para agregar, eliminar y obtener mascotas.

## Tecnologías
- C#
- ASP.NET Core
- Entity Framework
- JWT para autenticación

## Endpoints

### Nugets
- `Dotnetenv` 
- `JwtBearer` 
- `EntityFramework` 
- `EntityFramework.Design`
- `Pomelo.Mysql`
- `Swashbuckle`
- `Swashbuckle.Swagger`
